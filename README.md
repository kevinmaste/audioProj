### `Start`

You need to clone the project :

git clone https://github.com/kevinmaste/audioProj.git

**`run the front app`**
  1/ npm install
  2/ npm start
  
for the `Server` download it and run
  1/npm install
  2/nodemon app.js

Launch the projet the first page will be:
